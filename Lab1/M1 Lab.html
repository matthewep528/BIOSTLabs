Data description

A quantitative analysis of factors found in high-density lipoprotein (HDL) in 42 samples of human blood serum found three variables thought to be associated with HDL.  These variables were total cholesterol level, total triglyceride level, and a component called sinking pre-beta lipoprotein, which was coded as 0 if absent and 1 if present (dummy variable).  We will use these three variables to model HDL in this sample.

Data dictionary

Variable name (in data set) Variable description  
HDL
 High-density lipoprotein (HDL) value 
cholesterol
 Total serum cholesterol 
triglyceride
 Total triglycerides  
spbl
 Sinking pre-beta lipoprotein (0 if absent, 1 if present)

Directions

Complete each of the following sections. For questions asking you to fit a model or run a test, include all of your R code and output as well as your description/interpretation of the output. For questions asking for models or just interpretations, write your response in the text section where indicated in the template (.Rmd file). Each question is worth 1 point.

  1. Fit separate simple linear regression models for assessing the association of total cholesterol level (cholesterol), total triglyceride level (triglyceride) and sinking pre-beta lipoprotein (spbl) with HDL (HDL) (mg/dL). Write the estimated regression equation and interpret the slope coefficient for each of the three models.
  2. Using the separate models fit in part 1, test whether total cholesterol level, triglyceride level, and sinking pre-beta lipoprotein alone statistically significantly predict y. Interpret your answer.
  3. Write the theoretical (population) regression equation that would be used to test whether total cholesterol level, triglyceride level, and sinking pre-beta lipoprotein taken together statistically significantly improve the prediction of HDL. Interpret the coefficients in this model.
  4. Fit the model specified above and test if total cholesterol level, triglyceride level, and sinking pre-beta lipoprotein taken together statistically significantly improve the prediction of HDL. Interpret your answer.
  5. Fit a multivariable model and test whether sinking pre-beta (spbl) is associated with HDL after the combined contribution of cholesterol and triglyceride levels is taken into account (i.e., cholesterol and triglycerides are already in the model). Assume no interactions exist. State the appropriate null hypothesis for this test and interpret the results of the test.
  6. Suppose we are interested in evaluating if the association between cholesterol and HDL and the association between triglycerides and HDL (both) depend on the absence/presence of SPBL. Write down the model statement for the model you need to fit to assess this and interpret the coefficients in this model.
  7. Fit a multivariable model and test whether the interactions of cholesterol with SPBL and triglycerides with SPBL are simultaneously equal to zero in a model already containing the three main effects for cholesterol, triglycerides, and SPBL. State the null hypothesis of the test. Given the result of the test, what can you conclude about the relationship of HDL to both cholesterol and triglycerides (hint: remember what an interaction means)?
  8. For sinking pre-beta lipoprotein (spbl), compare the coefficient for spbl in the full model (in part 4) to the simple model in parts 1 and 2. Do you think there is confounding due to cholesterol and triglycerides? Explain.
  9. Assume straight-line models are appropriate for describing the relationship between HDL  and total cholesterol level for the absence and the presence of sinking pre-beta. Write out a single regression model that specifies two separate lines for the relationship between HDL and cholesterol, one for the absence of sinking pre-beta lipoprotein (spbl=0) and one for the presence of sinking pre-beta lipoprotein (spbl=1). (You can omit triglycerides from this model). Run code to fit this model.
  10. Fit separate models for each level of SPBL and compare these results to the results you got in part 9.
  11. Plot the observed data and the fitted lines from part 9 for each value of sinking pre-beta (either 2 separate graphs with the same y-axes or combined in one graph). What do you see?
  12. Fit the model you need to test for coincidence (equality) of the two lines in part 9. Perform the test and interpret.
  13. Fit the model you need to test for parallelism (equal slopes) of the two lines in part 9. Perform the test and interpret.
  14. Center the cholesterol and triglyceride variables at their mean value and refit the model from part 9. How do the models compare (which coefficients change and which stay the same)? Interpret the intercept in both models.
  15. In this dataset, the unit of measurement for cholesterol is mg/dL. Suppose we wish to convert this to mmol/L. To do so, we divide the value of cholesterol by 38.67. That is, x mg/dL=x mg/dL1 mmol/L38.67 mg/dL=x38.67 mmol/L

Apply this transformation and compare the simple linear regression model you fit in part 1 for cholesterol to a new model using the (uncentered) cholesterol value measured in mmol/L. What do you notice about the new regression coefficients?

 

Make sure to knit your document and turn in the HTML file with your code, output, and interpretations here!

Note: Please check your HTML output before submitting to make sure there aren't errors in your code that you aren't expecting! Questions with errors in the output will not receive full credit. Some common sources of errors:

  • Did you forget to include a line of code loading the packages (libraries) you need for your code in the markdown file (in a code chunk)?
  • Did you forget to load the data or to save the data with a name?
  • Did you run code in your console but forget to put it in a code chunk in your markdown file?
  • Did you make sure you're using the correct variable names (R is case-sensitive)?
